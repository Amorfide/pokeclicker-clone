# PokéClicker Clone

A simple **PokéClicker-inspired web game** built with **HTML, CSS, and vanilla JavaScript**.  

Click on Pokémon to reduce their health, attempt to catch them, and earn in-game currency. This project is designed as a learning exercise to practice core web development concepts while creating a fun, interactive game.

---

## Learning Goals

This project focuses on:
- Practicing DOM manipulation and event handling.
- Understanding game loops and state management in vanilla JS.
- Applying HTML/CSS/JS fundamentals.
- Gaining experience creating browser-based games from scratch.

---

## Features

- Click on Pokémon to reduce their health bar.
- Attempt to catch Pokémon when their health is low.
- Earn in-game currency and catch pokemon.
- Dynamic health bar updates and interactive feedback.
- Built with **vanilla JavaScript** and DOM manipulation.

---

## Technologies Used

- **HTML5** – Structure of the game interface.
- **CSS3** – Styling and layout of game elements.
- **JavaScript (Vanilla)** – Game logic, DOM manipulation, and event handling.

---

## How to Play

1. Open the game in your browser.
2. Click on the game screen to reduce the Pokémon's health.
3. Once the Pokémon's health is 0, the game will attempt to catch the Pokémon.
4. Earn currency for each Pokémon knocked out or caught.
5. Repeat and collect more Pokémon!
